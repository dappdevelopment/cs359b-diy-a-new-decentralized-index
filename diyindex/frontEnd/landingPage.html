DIY DAPP: The First Decentralized Rebalancing Index.

How does DIY Work? “DIY” is a dapp that allows the user to create a personalized Index Fund in which the user can select ERC20 tokens of choice and set parameters for rebalancing. User can also invest in funds that were started by other people and thematic indices. The dapp creates a smart contract which holds the users assets and allows the user to maintain control and custody of their assets. All trades (except rebalancing) will occur on a decentralized exchange (DEX), using the Radar Relayer API, for the purpose of this project. For now, rebalancing will be done in a centralized manner by us, but as a future implementation, the rebalancing will be also through a DEX.
